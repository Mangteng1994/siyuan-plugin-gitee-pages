# Pages Publisher — SiYuan Note to Pages

One-click publish SiYuan notes as **beautifully designed static HTML** to Gitee Pages or GitHub Pages.

## Design

Editorial/Magazine style with:
- Playfair Display + Source Serif 4 typography
- Warm light / dark dual theme (auto)
- Fade-up animations, responsive layout
- Refined blockquotes, code blocks, and tables

## Prerequisites

### GitHub Pages

1. Create repo named `yourusername.github.io`
2. Settings → Pages → Source: `main` → Save
3. `git clone` locally

### Gitee Pages (requires real-name verification in China)

1. Create a repo
2. Services → Gitee Pages → Start
3. `git clone` locally

## Usage

1. Restart SiYuan — icon appears in top bar
2. Click → select platform → fill in repo path & URL
3. Open a note → click icon → one-click publish
4. Visit your Pages URL after 1-3 min

## Features

- Dual platform: Gitee + GitHub, switch anytime
- Editorial-grade typography & dark mode
- Full Markdown: headings, lists, code, tables, quotes, images
- Auto Git: add → commit → push

## Notes

- Requires Git CLI
- GitHub Pages auto-deploys; Gitee Pages needs manual "Update" click
- Use with image hosting plugins for self-contained pages
